# Student_Analyzer# 📊 Student Analyzer (Pandas Project)

## 📌 Overview
This project analyzes student marksheets using **Python and Pandas**.  
It reads a CSV file containing student details and calculates useful statistics such as:
- ✅ Total number of students  
- ✅ Average marks  
- ✅ Highest and lowest scores  
- ✅ Topper of the class  

---

## 🛠️ Features
- Reads student data from a CSV file  
- Performs data analysis using Pandas  
- Displays summary statistics (mean, max, min)  
- Identifies the topper with the highest total marks
